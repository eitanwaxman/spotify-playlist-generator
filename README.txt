1. You will need a spotify developer account to use this program.
2. Set up the following as environment variables:

SPOTIPY_CLIENT_ID=***your client id****
SPOTIPY_CLIENT_SECRET=***your secret****
SPOTIPY_REDIRECT_URI=http://example.com

3. Log in to your spotify account when the Oauth opens
4. copy the *entire* URL from the redirect and paste in the console

5. Enter a date that you wish to create a playlist for.

Enjoy!
